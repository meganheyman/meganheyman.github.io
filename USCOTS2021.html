Indian Monsoon Example

Prompts and Solutions: This file contains prompts that may be used during class or in assignments to discuss spatio-temporal dependence in introductory topics of data exploration, univariate inference for 1 population mean, inference for 2 population means, and simple linear regression.

Dataset: This file contains the full dataset utilized within the Indian Monsoon Example. The data were obtained from the Open Government Data Platform India.

R Code: This file contains R code that may be used to create the plots shown within the Indian Monsoon Example.

Shapefile and Name Match: These files are used to create the choropleths of the Indian Monsoon precipitation in the example.

Some Other Examples

California County Water Usage: This assignment focuses on inference for two population means, specifically a paired data structure, and why the paired test would not be adequate with these spatio-temporal data. The assignment questions and solutions may be found here and the associated data are here. Note that the data are a small subset of the information provided by https://waterdata.usgs.gov/ca/nwis.
